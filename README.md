

This repository contains the source files for the **"Structural Equation Modeling with Onyx"** self-paced workshop.

## Workshop website

- Live workshop: https://brandmaier.github.io/lip2026-onyx-workshop

## Repository structure

- `sem-workshop/`: Quarto book source (`.qmd`), datasets, scripts, and images.
- `sem-workshop/_book/`: rendered HTML book output.

## Build locally

1. Install [Quarto](https://quarto.org/) and a recent LaTeX distribution (optional, only needed for PDF output).
2. From the repository root, run:

```bash
cd sem-workshop
quarto render
```

3. Open `sem-workshop/_book/index.html` in your browser.


The Onyx workshop is available here:

[](https://brandmaier.github.io/lip2026-onyx-workshop/sem-workshop/_book/index.html)
